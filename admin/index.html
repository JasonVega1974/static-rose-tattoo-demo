Set up this device again

One-time setup for this device. Paste the GitHub token once — it’s encrypted with your PIN and stored only in this browser, never in the site’s code. After this, you just enter the PIN.

How to create the token (1 min, do this once)
  1. Open github.com → Fine-grained tokens → Generate new.
  2. Resource owner: the GitHub account that owns this site’s repo.
  3. Repository access: Only select repositories → this site’s repo (see …).
  4. Permissions: Repository permissions → Contents → Read and write.
  5. Generate, copy the token, paste it above. It’s encrypted with your PIN and never leaves this browser.
